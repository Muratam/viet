# install
to execute `viet` command ...
```
### install nim
$ curl https://nim-lang.org/choosenim/init.sh -sSf | sh
$ echo 'export PATH=~/.nimble/bin:$PATH' >> ~/.bashrc
### install viet
$ git clone git@github.com:Muratam/viet.git
$ cd viet
$ nimble install
```
